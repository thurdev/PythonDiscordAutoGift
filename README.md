# Python Discord AutoGift

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

A simple program to automatically activate gifts from discord.



### Dependencies

| Plugin | README |
| ------ | ------ |
| Discum | https://pypi.org/project/discum/ |


### Installation

Open your favorite Terminal and run these commands.

First Step:
```sh
$ git clone https://github.com/thurdev/PythonDiscordAutoGift.git
```

Second Step:
```sh
$ cd 'cloned folder'
```

Third Step:
```sh
$ pip install discum
```

Fourth Step:
```sh
open main.py change the line 7 to your user token
```

Last Step:
```sh
$ py main.py
```

  - This code was made for studies only, if using it is at your own risk and will be against the discord's TOS.
